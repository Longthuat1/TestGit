


Ở vai trò người khởi tạo dự án .
1.Tạo remote Repository.
2. Tạo local Repository.
   Khởi tạo repo : git init 
   Kết nối remote repo với local repo : git remote add origin ' url remote repo '

3. Đẩy code lên remote repo :
   git add " tên file " hoặc git add . ( chọn tất cả các file thay đổi)
   git commit -m
